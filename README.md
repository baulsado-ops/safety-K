# 현장 위험성 체크리스트 생성기

제조업 사업장/건설현장 사진을 업로드하면, 작업 시작 전 위험성 체크리스트와 위험도 점수, 주의사항, 예방대책을 자동으로 생성하는 도구입니다.

## 주요 기능

- 사진 기반 위험요인 분석
- 작업 전 체크리스트 자동 생성(10개 이상)
- 종합 위험도 점수(0~100) 및 위험등급 산출
- 위험요인별 주의사항 + 예방대책 제시
- 리포트(`.md`) 다운로드

## 실행 방법

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
copy .env.example .env
# .env에 OPENAI_API_KEY 입력
streamlit run app.py
```

## 환경 변수

- `OPENAI_API_KEY`: OpenAI API 키 (없으면 데모 분석 모드)
- `OPENAI_MODEL`: 기본 `gpt-4.1-mini`

## 사용 방법

1. 현장 사진 업로드
2. 필요하면 추가 현장 정보 입력(야간작업/중장비 운행 등)
3. `위험성 분석 시작` 클릭
4. 결과 확인 및 리포트 다운로드

## 주의

- 본 도구는 안전관리자 판단을 보조하는 용도입니다.
- 실제 작업 허가/통제는 현장 책임자 및 법적 기준에 따라 최종 검토가 필요합니다.
